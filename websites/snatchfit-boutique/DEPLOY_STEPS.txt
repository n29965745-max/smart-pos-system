🚀 DEPLOY IN 3 SIMPLE STEPS (15 MINUTES)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: SETUP REQUIRED SERVICES (10 minutes)

1. MongoDB Atlas (Database)
   ✓ Go to: https://www.mongodb.com/cloud/atlas
   ✓ Create free account
   ✓ Create cluster
   ✓ Create database user
   ✓ Get connection string
   ✓ Copy: mongodb+srv://user:REDACTED_PASSWORD@cluster.mongodb.net/snatchfit

2. Stripe (Payments)
   ✓ Go to: https://stripe.com
   ✓ Create account
   ✓ Go to Developers → API Keys
   ✓ Copy test keys:
     - Publishable Key (pk_test_...)
     - Secret Key (sk_test_...)

3. GitHub (Code Repository)
   ✓ Go to: https://github.com
   ✓ Create account (if needed)
   ✓ Create new repository
   ✓ Name: snatchfit-boutique
   ✓ Make it public

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 2: PUSH CODE TO GITHUB (3 minutes)

Run these commands in your terminal:

cd websites/snatchfit-boutique

git init
git add .
git commit -m "Initial commit - production ready"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/snatchfit-boutique.git
git push -u origin main

Replace YOUR_USERNAME with your GitHub username!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 3: DEPLOY TO VERCEL (2 minutes)

1. Go to: https://vercel.com
2. Click "New Project"
3. Click "Continue with GitHub"
4. Select your snatchfit-boutique repository
5. Click "Import"
6. Add Environment Variables:

   MONGODB_URI = your_mongodb_connection_string
   JWT_SECRET=REDACTED
   NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY = pk_test_xxxxx
   STRIPE_SECRET_KEY = REDACTED_STRIPE_TEST
   NEXTAUTH_SECRET = your_nextauth_secret_key
   NEXTAUTH_URL = https://your-project.vercel.app
   NEXT_PUBLIC_APP_URL = https://your-project.vercel.app
   NEXT_PUBLIC_ADMIN_EMAIL = admin@snatchfit.com
   NEXT_PUBLIC_ADMIN_PASSWORD = admin123
   NODE_ENV = production

7. Click "Deploy"
8. Wait for build to complete (2-3 minutes)
9. Your site is LIVE! 🎉

🆘 TROUBLESHOOTING

Build fails?
→ Check package.json for correct versions
→ Run: npm install && npm run build

Environment variables not working?
→ Verify variable names match exactly
→ Check for typos
→ Redeploy after adding variables

Database connection error?
→ Verify MongoDB URI is correct
→ Check IP whitelist in MongoDB Atlas
→ Ensure database exists

Stripe not working?
→ Verify API keys are correct
→ Check Stripe account is active
→ Use test keys for testing
